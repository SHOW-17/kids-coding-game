# 🦄 あそびのもり

小学校1年生（6〜7歳）が、**文字を読まずに**遊びながら賢くなれる、こども向けのミニゲーム集（アーケード）。

オープニング画面（`index.html`）でゲームを選び、それぞれの遊びに入る。すべて **指タップ前提・ひらがな＋アイコンのみ・ビルド不要**（バニラJSの単一HTML）。

> **デザインテーマ「ねんどの森」** — ねんど（クレイ）／フェルトの質感の世界。マスコットはユニコーン（ブルー＝主役／グリーン＝はかせ／パープル＝ごほうび）。アイコンは絵文字でなく CSS図形、キャラは `assets/uni/` の画像で表現する。視覚言語の定義は [Design.md](Design.md) を参照（新画面はこれに従う）。

- **対象**：小1（漢字を読めない前提）。UIはひらがな＋アイコンのみ。
- **ねらい**：知的好奇心をくすぐり、非認知能力（集中・やりぬく力・試行錯誤・空間認識・創造性）を伸ばす。
- **やさしさ**：失敗しても叱らず、何度でも試せる。クリアは効果音・アニメ・紙吹雪で「できた！」を強く演出。
- **品質**：プロ品質を目標に、見た目・演出・手触りまで作り込む。

## あそべるゲーム

| ゲーム | 仕組み | 育てる力 |
|---|---|---|
| **プログラミング** | やじるしを並べて▶。命令どおりにユニコーンが動く | シーケンス・デバッグ・論理 |
| **まねっこ**（サイモン） | 光った順番を覚えてまねする | 記憶・集中・やりぬく力 |
| **きまり あそび** | 並びの「つぎは なに？」を当てる | 規則性の発見・推理 |
| **かたち づくり** | ピースを影にはめて形を作る | 空間認識・創造性・試行錯誤 |
| **ぴたごら** | 坂を置いて玉をゴールへ転がす | 因果・計画・試行錯誤 |

「プログラミング」のレベル構成や設計詳細は [CLAUDE.md](CLAUDE.md) を参照。

## 起動方法

ビルド不要。次のいずれかで `index.html` を開く。

```bash
# そのままブラウザで開く
open index.html            # mac
xdg-open index.html        # linux

# 簡易サーバーで配信（スマホ実機・Tailscale 等から見る場合）
python3 -m http.server 9090 --bind 0.0.0.0
# → http://<サーバーのIP>:9090/
```

`index.html` がアーケード（オープニング）。各ゲームは `kuma-programming.html` と `games/*.html`。

## ファイル構成

```
index.html              アーケード選択画面（オープニング／PWAのホーム）
kuma-programming.html    プログラミング（🏠でアーケードへ戻る。assets JS 不要の自己完結）

  katachi.html          かたちづくり（タングラム＝かげあわせ方式）
  pitagora.html         ぴたごら（たまころがし＝決定論グリッド）
assets/
  uni/                  ユニコーン画像（blue_/green_/purple_*。マスコット）

  fx.js                 共通エフェクト（紙吹雪・キラキラ・シェイク）
  save.js               共通セーブ（ゲーム別 localStorage）
```

> スタイルは各HTMLに**インライン**で持つ（共通の見た目ルールは [Design.md](Design.md)）。
> ※ `kuma-programming.html` というファイル名は保存キー互換のため維持（中身・テーマは「ねんどの森」に刷新済み）。

## 技術メモ

- バニラJS／ビルド不要。共通基盤（`assets/*`）の上に各ゲームを単一HTMLで実装。
- フォント：Google Fonts（Mochiy Pop One / M PLUS Rounded 1c）。
- 効果音：Web Audio で生成（音声ファイル不要、try/catch でガード）。
- 永続化：`localStorage`（ゲーム別の名前空間。使えない環境ではメモリに自動フォールバック）。
  アーケードの「🔄 きろくを ぜんぶ けす」で全消去。
- アクセシビリティ：`prefers-reduced-motion` を尊重。色だけに頼らない表現。
- 動作確認：`google-chrome-stable` + puppeteer-core で各ページの console error / 例外 / 横スクロールはみ出しを検査（`/tmp/test-games.js`）。

### デザイン素材について
ロゴ・ゲーム看板・背景・アイコンは CSS/SVG で「ねんど化」して実装（**絵文字は使わない**）。
マスコットは `assets/uni/` のユニコーン画像。追加・差し替えは [Design.md](Design.md) の配役ルールに従う。
