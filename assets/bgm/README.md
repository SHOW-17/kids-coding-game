# BGM 音源の置き場

ここに mp3 を置くと、そのファイルが BGM として優先再生される。
無い画面は `bgm.js` 内蔵の「やさしい合成ループ」が自動で鳴る（落ちない）。

## ファイル名（画面ごと）

| ファイル | 流れる画面 |
|---|---|
| `home.mp3`        | アーケード（index.html） |
| `programming.mp3` | プログラミング |
| `manekko.mp3`     | まねっこ |
| `kimari.mp3`      | きまりあそび |
| `katachi.mp3`     | かたちづくり |
| `pitagora.mp3`    | ぴたごら |

- ファイルを置いて画面を開くだけ。差し替えも上書きするだけ（コード変更不要）。
- うちの子がはまってる曲に随時入れ替えてOK。全画面同じ曲にしたいなら、同じ mp3 を各名前でコピーする。
- ループ前提なので、イントロ無し・きれいに繰り返せる音源だと自然。

## 注意（重要）

- **これらの mp3 はリポジトリにコミットされない**（`.gitignore` 済み）。
  このリポジトリは GitHub で公開されているため、市販曲（アナ雪など）の音源を
  push すると著作権侵害になる。音源は各自の手元にだけ置くこと。
- 配布・公開向けに BGM を同梱したい場合は、**自作曲またはライセンスがクリアな
  フリー音源**だけを使い、その時だけ `.gitignore` の該当行を外す。
