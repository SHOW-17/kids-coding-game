# BGM 音源の置き場

ここに mp3 を置くと、そのファイルが BGM として優先再生される。
無い画面は `bgm.js` 内蔵の「やさしい合成ループ」が自動で鳴る（落ちない）。

## いちばん簡単：全画面で同じ曲

`_default.mp3` を1つ置くだけで、全画面でその曲が流れる。

| ファイル | 流れる画面 |
|---|---|
| `_default.mp3` | **画面別ファイルが無い全画面の共通BGM** |

## 画面ごとに別の曲にしたいとき（任意）

下の名前で置くと、その画面だけ専用曲が優先される（無ければ `_default.mp3` → それも無ければ合成）。

| ファイル | 流れる画面 |
|---|---|
| `home.mp3`        | アーケード（index.html） |
| `programming.mp3` | プログラミング |
| `manekko.mp3`     | まねっこ |
| `kimari.mp3`      | きまりあそび |
| `katachi.mp3`     | かたちづくり |
| `pitagora.mp3`    | ぴたごら |

- 優先順位：`<画面名>.mp3` → `_default.mp3` → 合成ループ（`bgm.js` 内蔵）。
- ファイルを置いて画面を開くだけ。差し替えも上書きするだけ（コード変更不要）。
- ループ前提なので、イントロ無し・きれいに繰り返せる音源だと自然。

## 注意（重要）

- このリポジトリは GitHub で公開されているため、**置いてよいのは AI生成曲・自作曲・
  ライセンスがクリアなフリー音源だけ**。市販曲（アナ雪など）を push すると著作権侵害になる。
- 現在の `_default.mp3` は AI生成BGM（権利クリア）で、リポジトリにコミット済み。
  差し替えるときも権利のクリアな音源だけにすること。
